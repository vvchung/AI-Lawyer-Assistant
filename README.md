### üíª 30-Second Local Quick Start

Want to run this guardian on your own machine? Follow these simple steps:

### 1. Get the Code
Click the "Code" button in the top right, select "Download ZIP" and extract it, or use Git clone to copy the project to your local machine.

### 2. Install Dependencies
Open your Terminal or Command Prompt, navigate to the project folder, and run the following command to install the necessary packages:

```bash
npm install
```

### 3. Set Up API Key
You need a key to activate the AI.
1. Go to [Google AI Studio](https://aistudio.google.com/) to get a free API Key.
2. Create a `.env` file in the project's root directory.
3. Enter your key inside:

```env
API_KEY=YOUR_GOOGLE_API_KEY
```

### 4. Start the Application
Type the following in your terminal:

```bash
npm start
```

Open your browser and go to the URL shown in the terminal (usually `http://localhost:1234` or `http://localhost:3000`).

Now, your personal legal guardian is ready to serve you!
